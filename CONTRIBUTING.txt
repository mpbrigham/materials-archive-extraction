CONTRIBUTING
============

We welcome thoughtful, well-structured contributions. Please follow these principles:

1. DESIGN WITH INTENT
   - Match the clarity and minimalism of the existing prompts and workflows.
   - Include purpose and expected behavior in any new agent or prompt.

2. STRUCTURE PROMPTS CONSISTENTLY
   - Follow the Prompt Design and Prompt Style Guidelines.

3. VALIDATE BEFORE COMMITTING
   - Test workflows in n8n with placeholder data.
   - Validate all JSON outputs against the schema.

4. BE EXPLICIT
   - Avoid vague logic or ambiguous instruction.
   - Include comments where design decisions diverge from prior versions.

5. RESPECT THE KOAN
   - Preserve the tone and philosophy of the project.

Thank you for helping IMIS breathe more deeply.
