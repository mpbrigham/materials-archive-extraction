Required n8n Nodes for Deployment:

1. Webhook (POST, Binary Data = true)
   - Receives PDF upload from external app or frontend.
2. Extract From File (Operation: Extract From PDF)
   - Parses binary content into text.
3. Function Node (Pre-process)
   - Normalize and clean text (optional).
4. HTTP Request or OpenAI/Gemini Node
   - Sends structured prompt with OCR data to LLM.
5. Function Node (Validation)
   - Ensure LLM output matches JSON schema.
6. PostgreSQL or Google Sheets Node
   - Store validated metadata.
7. Slack/Email/Mattermost Node (Optional)
   - Notify success or error state.
8. Error Trigger (Global Error Handling)
   - Catch and route errors.

Note: Always test manually with PDF edge cases before production activation.
