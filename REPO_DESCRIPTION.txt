Intelligent Materials Intake System (IMIS)
------------------------------------------
A multi-stage, versioned pipeline architecture for extracting structured material metadata from industrial PDF documents.

Versions:
- V1_Linear_Flow: Prototype with linear logic, fallback-ready
- V2_Modular_Expansion: Modular agent orchestration with observability
- V3_Intent_Driven_Minimalism: Stateless-by-default, production-grade

Core features:
- Agent interface contracts
- Schema-first LLM prompting
- Feedback routing and CE fallback
- Full auditability and lifecycle logging
